# Notepad: Mapping tên ảnh thực phẩm với resource drawable
# Đặt file ảnh vào res/drawable và dùng tên không có phần mở rộng
# Ví dụ: ảnh "egg.png" trong drawable thì tên resource là "egg"
# Sử dụng đúng tên này khi tạo Food trong code hoặc nhập liệu

# Danh sách mẫu:
# Tên thực phẩm | Tên resource ảnh (không có .png/.jpg)
Trứng gà | egg
Sữa tươi | milk
Rau cải | vegetable
Thịt bò | beef
Cá hồi | salmon
# ... Thêm các dòng khác theo nhu cầu

# Nếu ảnh không hiển thị, kiểm tra lại tên file trong drawable và tên resource trong code phải giống nhau hoàn toàn (không có đuôi mở rộng, không phân biệt hoa thường)
